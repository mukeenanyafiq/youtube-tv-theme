# YouTube TV
YouTube Music with a style replicating YouTube TV playing Music with Lyrics, mostly on full-screen mode

That's all that I can say about it. Looks really nice though, and there's also a nice little extra features, if you don't mind

## Extra Features
- Changable layout placement (right or left)
- Side-bar, top-bar, and the play-bar all have transparent background
- Better Lyrics Shaders work really best when paired with this theme
